# Sales Assets

## Positioning

Use this sentence everywhere:

`We help local service businesses recover missed leads with instant reply, qualification, and booking automation.`

## Core promise

`If you are missing even a few good calls or form fills every month, this usually pays for itself almost immediately.`

## Offer stack

- missed-call text-back
- instant website lead capture
- AI qualification
- calendar handoff
- owner alerts
- weekly lead summary

## Objection handling

### "We already answer the phone"

`That helps during business hours. This is for the leads that come in after-hours, during busy periods, or when the office line gets missed.`

### "We already have a contact form"

`Forms are passive. The point here is instant response and qualification, so good leads do not cool off while they wait.`

### "We are not ready for a big software project"

`This is not a big software project. It is a lightweight install that sits on top of your current site and phone workflow.`

### "We need proof first"

`That is exactly why I can show a niche-specific demo and start with a narrow pilot if needed.`

## Pilot offer

Use this when buyers hesitate on the full setup fee:

`I can install a narrow version for 7 days, focused on missed-call text-back and website lead capture, and if you like what you see we roll it into the full setup.`

## Call outline

1. Ask how many calls or web leads they get in a typical week.
2. Ask what happens when nobody answers right away.
3. Show the demo flow.
4. Tie one recovered job to the setup fee.
5. Offer install timing and the payment link.
